# מדריך כתיבה להמרות

חל על: דף הבית, hubs, `/book`, בלוג, הודעות WA, yakir-closer.

## סגנון

- מקף רגיל `-` בלבד. לא em-dash `—` ולא `–`.
- משפטים קצרים. פסקאות עד 3 שורות.
- גוף ראשון: "אני" / "אתה".
- בלי סימני קריאה מיותרים. בלי AI-speak.

## CTA

- תמיד: **תוצאה + מחיר** (לא "שלח" / "צור קשר").
- דוגמה: "אני רוצה הצעה לפודקאסט מ-950 ₪ (לפני מע״מ)".
- מחיר סופי כולל מע"מ בולט בנקודת החלטה.

## נוסחת מסגור

```
[כאב] → [פתרון מדיד] → [מחיר שקוף] → [עלות חלופית]
```

מקור יחיד: `lib/data/pricing-catalog.ts`. אותו מחיר באתר, WA, closer ובלוג.
