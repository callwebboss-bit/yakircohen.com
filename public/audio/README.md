# קבצי אודיו - דוגמאות "לפני / אחרי"

התיקייה הזו מכילה את קבצי ההדגמה שמתנגנים בנגני "לפני/אחרי" באתר.
**הקבצים האלה לא קיימים עדיין - צריך להעלות אותם לכאן בדיוק עם השמות הבאים.**

כל קובץ: MP3, באורך 20-60 שניות, מומלץ עד 2MB.

| שם הקובץ (חובה מדויק) | מה מקליטים | מופיע בעמוד |
|------------------------|-------------|-------------|

## איך מעלים

1. גוררים את קובץ ה-MP3 לתוך התיקייה הזו (`public/audio/`).
2. מוודאים שהשם **זהה בדיוק** לטבלה (אותיות קטנות, מקפים, סיומת `.mp3`).
3. שומרים - הנגן באתר יזהה את הקובץ אוטומטית, בלי שינוי קוד.

> טיפ: עד שמעלים את הקבצים, הנגנים מציגים הודעת "יועלה בקרוב" עם קישור לוואטסאפ.
